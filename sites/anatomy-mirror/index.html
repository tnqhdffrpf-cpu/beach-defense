Anatomy Mirror

Live front-camera skeletal visualization inspired by the reference post: MediaPipe tracks your pose, and Three.js renders an x-ray chest panel anchored to your torso in real time.

This version is built as a static browser app for GitHub Pages and iPhone Safari. It recreates the same core interaction and visual effect, while using original rendering and UI code.

Works best in portrait mode with your chest and both shoulders visible. Camera access requires HTTPS, which GitHub Pages provides.
