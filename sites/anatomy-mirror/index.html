Anatomy Mirror

Live front-camera anatomy effect inspired by the reference post: MediaPipe tracks your hands and pose, and Three.js renders a hand-framed x-ray window in real time.

This version is built as a static browser app for GitHub Pages and iPhone Safari. It uses a real chest x-ray image and a moving square frame controlled by your thumbs and index fingers.

Works best in portrait mode with your chest, shoulders, hands, thumbs, and index fingers visible. Camera access requires HTTPS, which GitHub Pages provides.
